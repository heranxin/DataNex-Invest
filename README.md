# DataNex-Invest
本项目为一款基于Python与多模态AI模型的智能股价预测系统。传统股价预测仅依赖历史行情数据，预测维度单一、准确率有限。本系统创新性融合文本、图像、时序数据多模态信息，通过爬虫与公开财经接口获取股票K线时序数据、K线图表图像、财经新闻、上市公司公告等多维数据。 系统利用CNN模型提取K线图像特征，借助大语言模型完成新闻舆情情感分析与文本解读，结合LSTM时序模型挖掘股价涨跌规律，通过多模态融合算法综合研判市场趋势，最终实现股票短期涨跌趋势预测、概率分析与可视化结果展示。系统配备简易交互界面，支持数据查询、模型预测、结果展示与历史回测，能够有效辅助用户进行金融行情研判，具备较强的实用性与创新性。
